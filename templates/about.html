Release Notes
What's new in v1.0
Version 1.0 — Initial Release
Stable
Released 2025 · First public version
The first stable release of StudentAI. This version establishes the core platform — AI-powered academic Q&A, secure user authentication with email OTP, voice input and output, a live audio visualizer, and a clean dark UI designed for long study sessions.
NewGroq LLaMA 3.1 Integration — Ultra-fast academic answers powered by LLaMA 3.1-8b-instant with a custom student-focused system prompt
NewEmail OTP Authentication — Secure signup and password reset verified through Gmail OTP with 10-minute expiry
NewVoice Input — Speak questions directly using your browser's Speech Recognition API with live word highlighting
NewText-to-Speech Output — AI responses are spoken aloud with the audio visualizer reacting in real time
NewLive Audio Visualizer — Animated bar visualizer that pulses when the AI speaks, idle when silent
NewChat History — Full conversation history with user and AI bubbles saved to SQLite database per session
NewDark UI Design System — Consistent dark theme across all pages — login, signup, OTP verify, forgot password, chat, home, about
NewSmall Talk Handler — Instant replies for greetings without wasting API calls
ImprovedSession Security — SHA-256 hashed passwords, Flask session protection, unauthorized request blocking
FixedTranscript Overflow — Scrollable transcript area prevents long AI responses from overflowing the screen
